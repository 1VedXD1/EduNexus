The <head> Element

The <head> element contains meta-information about the document, such as its title, character set, stylesheets, and scripts.

<head>
    <meta charset="UTF-8">
    <title>Page Title</title>
    <link rel="stylesheet" href="EduNexus.css">
</head>

The <title> Element

The <title> element defines the title of the document, which is displayed in the browser's title bar or tab.

<title>EduNexus - Learn Web Development</title>

The <link> Element

The <link> element defines the relationship between the current document and an external resource, typically used to link to external stylesheets.

<link rel="stylesheet" href="EduNexus.css">

The <meta> Element

The <meta> element represents metadata about an HTML document, such as character encoding, author, and viewport settings.

<meta charset="UTF-8">

The <nav> Element

The <nav> element defines a set of navigation links. It is typically used for primary navigation within the website.

<nav>
    <ul>
        <li><a href="home.html">Home</a></li>
        <li><a href="about.html">About</a></li>
    </ul>
</nav>

Try it Yourself

The <form> Element

The <form> element represents a form used to collect user input. It can contain various input elements like text fields, checkboxes, radio buttons, and more.

<form action="/submit" method="post">
    <label for="name">Name:</label>
    <input type="text" id="name" name="name" class="input-text">
    <input type="submit" value="Submit" class="submit-button">
</form>

Try it Yourself

The <input> Element

The <input> element is used to create interactive controls for forms. Common attributes include type, id, name, and class.

<input type="text" id="username" name="username" class="input-text">

Try it Yourself

The <div> Element

The <div> element is a block-level container used to group elements for styling purposes. It is one of the most commonly used elements in web development.

<div class="container">
    <p>This is a paragraph inside a div.</p>
</div>

Try it Yourself

The <span> Element

The <span> element is an inline container used to group text or inline elements. It is often used for applying styles or scripts to parts of text.

<p>This is a <span class="highlight">highlighted</span> word.</p>

The <h1> Element

The <h1> element represents a top-level heading in an HTML document. It is typically used as the main title or heading for a section of content.

<h1>Welcome to EduNexus</h1>

The <p> Element

The <p> element represents a paragraph of text. It is a block-level element typically used for structuring text content in a document.

<p>This is a paragraph of text.</p>

The <a> Element

The <a> element represents a hyperlink, linking to another page or resource. Common attributes include href, target, and rel.

<a href="https://www.example.com">Visit Example.com</a>

The <button> Element

The <button> element represents a clickable button, which can be used to submit forms or trigger events in the browser.

<button type="submit">Submit</button>
